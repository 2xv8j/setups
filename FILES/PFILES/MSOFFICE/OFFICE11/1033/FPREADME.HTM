Internet Explorer Enhanced Security Configuration issues

Microsoft Windows Server 2003 is installed with Microsoft Internet Explorer Enhanced Security Configuration enabled by default. These settings can prevent some Web sites— including sites created by using Microsoft Office FrontPage 2003— from being displayed properly or performing as expected.

For more information, visit the Microsoft Product Support Services site at http://support.microsoft.com.
